# NexaCard Secondary Development Notice

NexaCard Admin is an independently maintained secondary-development branch of the open-source Dujiao-Next admin frontend.

This repository uses NexaCard branding, UI direction, release tags, and deployment documentation. Do not point update prompts, release links, or production documentation back to the upstream Dujiao project. NexaCard release coordination should use:

- Admin releases: https://github.com/NexaCard/admin/releases
- API releases: https://github.com/NexaCard/API/releases
- User frontend: https://github.com/NexaCard/user
- Documentation: https://github.com/NexaCard/docs
