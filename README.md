# NexaCard Admin

NexaCard Admin is the management console for operating products, orders, users, payment channels, and system settings.

## Secondary Development Notice

NexaCard Admin is an independently maintained secondary-development branch based on the open-source Dujiao-Next admin frontend. NexaCard uses its own UI, release tags, update channels, and documentation under the `NexaCard` GitHub organization. See [NOTICE.md](./NOTICE.md).

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Tailwind CSS

## What This App Does

- Dashboard and operational analytics
- Product/category/content management
- Order and fulfillment management
- Payment channel configuration
- Admin role and permission management

## Quick Start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
